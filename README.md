# Desafio @CrossJoin

## Objective

Develop a C# application to manage Companies, Leads, Proposals, and Products. The system must support creating and evolving these entities, validating required fields, and enforcing dynamic business rules.


## How to Build

```bash
cd desafio@CrossJoin
dotnet build
```

## How to Run the Application

```bash
cd desafio@CrossJoin
dotnet run
```

## How to Run Tests

```bash
# Run all tests
cd desafio@CrossJoin/desafio@CrossJoin.Tests
dotnet test

```
